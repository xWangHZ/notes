## ls命令

功能描述：**显示目录文件**

参数：

* **-a：全部文件，包括隐藏文件(开头为.的文件)一起列出来**
* **-l：显示详情信息(可简写为ll)**
* **-d：只看当前文件中的目录信息**
* **-R：向下递归查看目录信息**

## cd命令

功能描述：**移动到对应位置**

命令格式：**cd 参数 位置**

几个特殊目录

* **.	此层目录**
* **..  上一层目录**
* **-   前一个工作目录**
* **~  目前用户身份所在的家目录**

## pwd命令

功能描述：**显示出目录**

参数：

* **-P：显示链接文件指向的位置**

## mkdir命令

功能描述：创建一个目录

命令格式：**mkdir 参数 目录名称**

参数：

* **-p：向上递归创建目录**

## rmdir命令

功能描述：**删除空目录**

命令格式：**rmdir 参数 目录**

参数：

* **-p：向上递归删除目录**

## cp命令

功能描述：**复制文件或目录**

命令格式：**cp 参数 原文件 目标文件**

参数：

* **-p：连同文件的属性一起复制过去，而非使用默认属性(备份常用)**
* **-r：复制该目录下所有的子文件和文件**
* **-a： 保留链接、文件属性，并复制目录下的所有内容。**

## rm命令

功能描述：**删除文件或目录**

命令格式：**rm 参数 文件或目录**

参数：

* **-f：强制执行**
* **-i：在删除前会询问使用者是否操作**
* **-r：递归删除，常用于目录的删除**



## mv命令

功能描述：剪切文件或改名

参数：

* **-f：强制，如果文件已存在不会询问直接覆盖**
* **-i：若已存在，则询问**
* **-u：若文件已存在，且文件较新时，才会更新**



## touch命令

功能描述：创建空文件



## echo命令

```shell
echo Hello #打印文本
echo 文本内容>文件名	#打印文本到文件
```



## 查看文件

* **cat：从第一行开始显示文件内容**
* **tac：从最后一开始显示**
* **nl：显示的时候，输出行号**
* **more：一页一页的显示文件内容**



## cat命令

功能描述：显示文件内容

```sh
cat >文件名
输入文件内容
ctrl+C退出
```



```shell
cat 文件1 文件2>新的文件
#将两个文件合成一个新的文件
```



## more命令

功能描述：分页显示内容

语法：

* 空格或f翻页
* Enter换行
* q或Q退出
* /字符串：向下检索字符串

## less命令

功能描述：分页显示内容

语法：

* [pagedown]向下翻页
* [pagedup]向上翻页

## head命令

功能描述：显示文件前面几行

参数：

* **-n：指定行数**



## od命令

功能：非纯文本输出

参数：

* **-a：默认字符输出**
* **-c：ASCLL字符输出**
* **-d：十进制输出**
* **-x：十六进制输出**
